### Description
This snapshot represents the current state of the header layout. The header includes a logo, title, and navigation menu with dropdown functionality. The layout is responsive and adjusts for mobile devices. Key features include:

1. Full-width header with left-aligned logo and title, and right-aligned navigation.
2. Dropdown menus for Sports, Finance, Weather & Travel, and News sections.
3. Responsive design that stacks elements vertically on smaller screens.
4. Increased text size (by 10%) and added padding for better readability and spacing.
5. Hover effects on navigation links and dropdown items.
6. Custom styling for the logo, title, and navigation elements.

The Header component uses the Navigation component to render the menu items, providing a clean separation of concerns. The CSS is organized to handle both desktop and mobile layouts, ensuring a consistent user experience across devices.
